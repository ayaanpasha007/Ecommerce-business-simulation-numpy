# E-Commerce Business Simulation (NumPy Mini Project)

This project simulates 30 days of e-commerce business data using NumPy probability distributions.

## Features
- Daily customers (Poisson distribution)
- Purchase amount simulation (Normal distribution)
- Product category selection (Multinomial distribution)
- Delivery time estimation (Poisson distribution)
- Waiting time between orders (Exponential distribution)
- Product quality check (Binomial distribution)

## Requirements
- Python 3.x
- NumPy

## How to Run
```bash
python ecommerce_simulation.py
```

## Purpose
This project demonstrates how probability distributions are used in data science,
machine learning simulations, and business analytics.
